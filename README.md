# Cognevance Job Portal

A full-stack online job portal where employers can post jobs and job seekers can browse, search, and apply to them. Built with Django REST Framework and React.

## Features

- User authentication with JWT (register/login)
- Role-based access: Employer and Job Seeker
- Employers can post, edit, and delete job listings
- Employers can view and manage applicants (accept/reject/review)
- Job seekers can search and filter jobs by title, location, and type
- Job seekers can apply to jobs and track application status
- Responsive dashboard for both roles

## Tech Stack

**Backend:** Django, Django REST Framework, Simple JWT, SQLite (dev)
**Frontend:** React, React Router, Axios
**Auth:** JSON Web Tokens (JWT)
